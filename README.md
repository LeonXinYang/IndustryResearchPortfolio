您好！首先感谢您耐心阅读我的简历并看到这里！以下内容是我过往实习中关于行业研究报告撰写的内容，不涉及任何涉密内容。

我并不擅长于吹捧自己，而过往实习中的学习经历和感受也并非三言两语可以厘清，我更多是想实实在在想告诉您我实习里做过什么而并非虚度时光，
因此我把过往实习中，由自己100%独立完成的内容均放入此文件夹。
